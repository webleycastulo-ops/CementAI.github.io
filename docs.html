系统简介

CementAI 是一个基于贝叶斯优化的水泥浆配方智能研发平台。其核心思想是将传统的"试错式"实验设计转化为AI驱动的闭环优化过程：AI根据历史实验数据预测最有潜力的下一组参数，自动提交给性能预测网站，读取结果后更新内部模型，循环迭代直至找到最优配方。

💡 与随机搜索相比，贝叶斯优化通常能在 20次实验内 达到随机搜索100次才能达到的优化效果。

平台完全运行在浏览器中，无需安装任何软件，支持所有主流浏览器（Chrome、Edge、Firefox、360、QQ浏览器等）。

快速上手

  • 打开优化系统
    点击导航栏"优化系统"或首页"立即体验优化"按钮，进入优化界面。
  • 设置参数范围
    在左侧面板中，系统已预设合理的初始参数值。你可以直接使用默认值，也可以根据实验经验调整各参数。如需固定某个参数（例如固定温度），点击参数右侧的锁定按钮🔓。
  • 选择优化算法
    推荐使用贝叶斯优化（BO），适合样本量少、实验成本高的场景。如需探索更大的参数空间，可选择"混合策略"。
  • 配置优化参数
    设置最大实验次数（建议20-30次），填写总预算上限（系统会自动在预算内搜索），调整各目标权重（加大抗压强度权重以优先提升强度）。
  • 启动优化
    点击▶ 启动优化，系统开始自动运行。中央日志区域会实时显示AI决策过程和预测结果，右侧会更新最优配方和性能雷达图。
  • 导出与分析
    优化完成后，点击"导出CSV"保存实验记录，然后进入"数据分析"页面上传CSV，生成收敛曲线、帕累托前沿等可视化图表。

优化目标与权重

系统支持四个性能指标的加权多目标优化，综合评分计算公式为：

# 各指标归一化到 [0, 1]，再按权重加权平均
s_score = min(1.0, 强度 / 80.0)           # 80 MPa 为满分
p_score = max(0.0, 1 - 渗透率 / 0.5)     # 越低越好，0.5 mD 为 0 分
t_score = max(0.0, 1 - |稠化时间 - 150| / 150)  # 目标 150 min
f_score = max(0.0, 1 - 失水量 / 150)     # 越低越好

综合评分 = (w₁·s + w₂·p + w₃·t + w₄·f) / (w₁+w₂+w₃+w₄) × 100
💡 若主要目标是最大化抗压强度，建议将抗压强度权重设为 0.6-0.8，其余权重相应减小。

成本约束模型

每次实验的成本由固定成本和变动成本两部分组成：

单次成本 = 100 + Σ (参数相对用量 × 成本权重 × 20)

# 例：Ca/Si=1.5, Al/Si=0.3, W/C=0.4 时
成本 ≈ 100 + (1.5-0.3)/(2.0-0.3)×2×20 + 0.3/0.5×3×20 + (0.4-0.3)/(0.7-0.3)×1×20
     ≈ 100 + 28.2 + 36.0 + 5.0 ≈ 169.2 元

系统会在每次提议参数时检查剩余预算，若成本超出则自动跳过并重新采样。

贝叶斯优化原理

高斯过程代理模型

贝叶斯优化使用高斯过程（GP）对真实性能函数建立概率代理模型。每次实验后，GP会更新对整个参数空间的预测分布（均值μ和方差σ），使AI"知道"哪些区域有潜力、哪些区域不确定性高。

期望改进（EI）采集函数

基于GP预测，计算每个候选点的期望改进值，平衡"利用已知好区域"和"探索未知区域"的权衡：

EI(x) = (μ(x) - f_best - ξ) · Φ(Z) + σ(x) · φ(Z)
        其中 Z = (μ(x) - f_best - ξ) / σ(x)
        ξ = 探索系数（越大越倾向探索新区域）
⚠️ 前5次实验使用拉丁超立方采样均匀初始化参数空间，之后才切换为基于GP的EI优化。请勿在前5次就判断收敛。

AI对话分析

优化界面底部的对话框直接连接 Claude AI，系统会将当前最优配方、所有实验结果和迭代状态作为上下文传递给AI，你可以用自然语言询问：

"当前Ca/Si=1.24为什么比1.0强度更高？从C-S-H理论解释"
"如何在不增加成本的前提下进一步降低渗透率？"
"分析一下参数敏感性，哪个参数对强度影响最大？"
"当前稠化时间偏短，应该调整哪个参数？"
💡 AI对话功能需要访问 Anthropic API，请确保网络可以连接 api.anthropic.com。

Python 后端集成

如果你的性能预测是一个网页表单，可以下载我们提供的 Python 版本，使用 Selenium 自动填表：

from optimizer import CementOptimizationLoop

loop = CementOptimizationLoop(
    url="http://101.43.178.48/",
    total_budget=5000.0,   # 总预算（元）
    max_iterations=30,     # 最大实验次数
    headless=True,          # 无界面运行
    locked_params={},         # 锁定参数：{"temperature": 180}
)
best = loop.run()
print(f"最优评分: {best.scoreVal:.4f}")
print(f"最优配方: {best.params}")

使用前需要先运行 web_inspector.py 检测网站的表单字段名，然后修改 FIELD_SELECTORS 和 RESULT_SELECTORS 适配你的网站。详见下载包中的 README.md。
